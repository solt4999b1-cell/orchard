⚙️ OCR 엔진 설정

Google Drive OCR을 연결하면 한글 인식률이 크게 향상됩니다.
Google Apps Script 웹앱으로 무료 배포할 수 있습니다.

배포 방법 요약
① script.google.com → 새 프로젝트
② 코드 붙여넣기 (Code.gs) — 첨부된 파일 참고
③ 좌측 서비스(+) → Drive API 추가
④ 배포 → 웹앱 → 액세스: 모든 사용자
⑤ 발급된 URL을 위 입력란에 붙여넣기
